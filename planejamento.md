# Planejamento - SQL Queries Portfolio

## Schema planejado (para criar no SQL Online)

### Tabela: vendas
- id (INT, PRIMARY KEY)
- produto (VARCHAR)
- categoria (VARCHAR)
- quantidade (INT)
- valor (DECIMAL)
- data_venda (DATE)

### Tabela: clientes
- id (INT, PRIMARY KEY)
- nome (VARCHAR)
- cidade (VARCHAR)

## Queries que vou desenvolver
1. SELECT com WHERE
2. JOIN (INNER e LEFT)
3. GROUP BY com COUNT e SUM
4. Subquery com IN
5. ORDER BY e LIMIT
